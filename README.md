# 🎬 Aiedio - 人工智能视频生成平台 (开发协作大本营)

欢迎来到 Aiedio 项目！这是咱们 5 人小队从零开始构建“自动化视频生成流水线”的主战场。
如果你是**刚进组、电脑上什么都没有的新手**，请**严格按照以下步骤**，一步步把开发环境搭建起来，绝对能顺畅跑完！

---

## ⚡ 研发环境“从零开始”起步指南 (小白协作者专属)

### 第一步：安装“必备三大件”软件
如果你那是一台刚买回来的新电脑，请先去官网下载并无脑“下一步”安装这三个软件：
1. **[Git (版本控制器)](https://git-scm.com/downloads)**  
   *下载并一路默认回车安装。这是让咱们能同步代码的神器。*
2. **[Visual Studio Code (写代码的编辑器)](https://code.visualstudio.com/)**  
   *所有老司机都在用的超强写代码软件，咱们统一用这个。*
3. **环境运行时（根据你的工种二选一）**：
   - 如果你是**后端、引擎、爬虫(backend/core/crawler)**的同学：请去安装最新版 **[Python (建议 3.10+)](https://www.python.org/downloads/)**。*(安装时必选底部的 "Add Python.exe to PATH" 框！)*
   - 如果你是**前端(client)**的同学：请去安装 **[Node.js](https://nodejs.org/en)**。

---

### 第二步：注册 GitHub 并“接受入驻邀请” (核心！)

1. 打开浏览器登录 **[GitHub 官网](https://github.com/)**。
2. 点击右上角的 Sign up 注册一个你个人的账号名称（**用你最喜欢的英文/数字拼音，不要有空格哦**）。
3. 注册完毕后，把你的 **[GitHub 账号名称]** 发到咱们的微信小群。
4. **⚠️ 极其重要**：等待组长 keeker765 在后台添加你。加完之后，你需要登录你注册时用的邮箱，或者直接登录 GitHub 首页 看顶部黄条，**务必点击 "Accept Invitation" (接受邀请)**。不点这个，你后续永远无法推送代码！

---

### 第三步：鉴权登录与克隆代码 (将云端搬回家)

确认接受邀请后，接下来要把项目拉到你的电脑上。克隆的瞬间系统就会**要求你进行鉴权（登录）**：
1. 在你的电脑里找个足够好、英文路径的盘（例如 D:\py_projects\）。
2. 在这里打开你刚安装好的 **VS Code** 软件。
3. 按下键盘顶部的 Ctrl + ~（或者点击顶部菜单栏的 Terminal -> New Terminal），呼出底部黑框。
4. 在其中**敲下这行克隆代码并回车**：
   `ash
   git clone https://github.com/keeker765/aiedio.git
   `
5. **🔥 关键的登录鉴权出现**：当你敲下回车后，屏幕正中央必定会弹出一个叫做 **Git Credential Manager** 的白框！
   - **不要慌！** 
   - **直接点击弹窗上的 "Sign in with your browser" (在浏览器中登录)**。
   - 网页会自动打开，点击绿色的 **"Authorize / Approve"** 按钮。
   - 网页提示成功后关闭它，此时你的终端就开始刷刷刷下载代码了。这代表你**登录鉴权彻底成功**，以后这台电脑永远都不用再输密码了！
6. 下载彻底完成后，VS Code 顶部菜单栏点击 File -> Open Folder，选中刚刚下载下来的名为 iedio 的文件夹并打开！

---

### 第四步：自报家门（配置本地提交凭证）

顺利打开 iedio 文件夹后，为了让 Git 历史里能显示代码是“谁”写的，你还需要配置两行文本（这只是署名，不是登录）。
再次按下 Ctrl + ~ 打开终端（确保当前路径已经在 aiedio 文件夹内了），依次敲下这两行代码（注意把大括号里的内容换成你自己的）：
`ash
git config --global user.name "填写你的真实拼音全拼"
git config --global user.email "填写你注册Github的真实邮箱"
`
*(敲完没有任何报错提示，就代表你已经是一名正式挂牌的正规军了！)*
